# Tech Stack - Genie

## Frontend
- **Framework:** React 19
- **3D Rendering:** Three.js with `@react-three/fiber` and `@react-three/drei`
- **Styling:** Tailwind CSS v4 (with Glassmorphism focus)
- **State Management:** Zustand
- **Build Tool:** Vite
- **Language:** TypeScript

## Backend
- **Runtime:** Node.js
- **Real-time Communication:** Socket.io

## AI Pipeline Integrations
- **Image Generation:** Nano Banana API
- **3D Reconstruction:** Marble World Labs API (Gaussian Splatting)
- **Visual Enhancement:** Decart Realtime API

## Infrastructure & Deployment
- **Hosting:** GitHub Pages (Frontend)
- **Version Control:** Git
